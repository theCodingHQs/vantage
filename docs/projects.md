# Projects & Task Management

Vantage lets you manage project deliverables and budget margins directly alongside your client profiles.

---

## 1. Project List

- Lists all active, planning, completed, or cancelled projects.
- Card metrics show how many hours have been logged on each project and the billing style (fixed contract value vs hourly rate).

---

## 2. Project Detail Drawer

Click on a project to open its details drawer:

- **Budget Tracker**: Calculates your logged time value against the total contracted budget. The progress bar updates live.
- **Scope**: Review project descriptions and deadlines.
- **Task Checklist**: Uses a Kanban board (To Do, In Progress, In Review, Completed). Drag and drop tasks to update progress. Click "+ Add Task" to insert a new checkbox.
- **Logged Time**: Shows all time logs recorded for this project.
